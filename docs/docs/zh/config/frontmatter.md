你可以在页面的 frontmatter 配置以下选项设置页面相关信息。

## title

- 类型：`string`
- 必填：否

当前页面的内容标题，默认为 Markdown 文件中的第一个 `h1` 标题。

## description

- 类型：`string`
- 必填：否

当前页面内容描述。

## author

- 类型：`string`
- 必填：否

当前页面的作者。

## date

- 类型：`DateString`
- 必填：否

写作时间，格式: `YYYY-MM-DD`。

## categories

- 类型：`string | string[]`
- 必填：否

分类。

## tags

- 类型：`string | string[]`
- 必填：否

标签。
