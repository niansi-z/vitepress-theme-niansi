## 用法

可以像这样在一个 markdown 文件中包含另一个 markdown 文件，甚至是内嵌的。

:::tip 提示
也可以使用 `@`，它的值对应于源代码根目录，默认情况下是 VitePress 项目根目录，除非配置了 `srcDir`。
:::

使用方式参阅 [VitePress → 包含 markdown 文件](https://vitepress.dev/zh/guide/markdown#markdown-file-inclusion)。
