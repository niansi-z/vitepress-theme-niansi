主题可以为你添加提示、注释、信息、注意、警告和详情提示容器的样式支持。

## 演示

#### 输入

```md
::: info
信息容器。
:::

::: tip
提示容器。
:::

::: warning
警告容器。
:::

::: danger
危险容器。
:::

::: details
详情容器
:::
```

#### 输出

::: info
信息容器。
:::

::: tip
提示容器。
:::

::: warning
警告容器。
:::

::: danger
危险容器。
:::

::: details
详情容器
:::
