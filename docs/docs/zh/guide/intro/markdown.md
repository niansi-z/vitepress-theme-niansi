VitePress 主要从 Markdown 文件生成页面。因此，你可以使用它轻松生成博客站点。

你需要创建并编写 Markdown，以便 VitePress 可以根据文件结构将他们转换为不同的页面。

## Markdown 配置

VitePress 通过 Frontmatter 为每个 Markdown 页面引入配置。

> [!IMPORTANT] Frontmatter
> Frontmatter 是 VitePress 中很重要的一个概念，请阅读 [VitePress Frontmatter](https://vitepress.dev/zh/guide/frontmatter) 了解详情

## Markdown 扩展

VitePress 使用 [Markdown-It](https://github.com/markdown-it/markdown-it) 来解析 Markdown 内容，因此可以借助于 Markdown It 插件来实现语法扩展。

- VitePress 扩展

  为了丰富写作，VitePress 对 Markdown 语法进行了一些扩展，请阅读 [VitePress Markdown扩展](https://vitepress.dev/zh/guide/markdown) 了解详情。

- 自行扩展

  你可以通过寻找合适的 Markdown It 插件来扩展 Markdown 语法。
