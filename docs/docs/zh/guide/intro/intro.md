## 主题初衷

VitePress 在一定程度上扩展了 Markdown 语法，但仍然缺少一些常用功能，同时默认主题提供的一些功能较弱或缺失，
如图片预览。同时 VitePress 默认主题不适合博客网站，这种情况下 `vitepress-theme-niansi` 就诞生了。

## 设计目标

:::info 最小化配置
如果可能，所有功能都会尝试使用默认值，因此可以在另配置或最小配置下使用。

这有助于减少迁移或学习成本，同时直接享受他们的便利。
:::

:::info 改进的布局
主题界面已经完全重构，以提供可定制和漂亮的布局。
:::

## 页面元数据

- 页面信息包括：
  - 作者
  - 书写日期
  - 预计阅读时间
  - 标签和类别

## 博客支持

该主题支持博客，自动摘录生成，自动摘要。

- 支持类别和标签、归档。
- 全新的博客主页
- 提供评论功能以便可以与访客交流
