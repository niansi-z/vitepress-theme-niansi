可以在主题选项控制博客中各页面路径。

博客提供的默认路径如下，如果他们与你的已有路径发生冲突，并且你不想调整自己的路径，
可以对他们进行修改。

| **配置项**        | **描述** | **默认路径**       |
|----------------|--------|----------------|
| `archivesPath` | 归档页面   | `/archives/`   |
| `categoryPath` | 分类页面   | `/categories/` |
| `tagPath`      | 标签页面   | `/tags/`       |
