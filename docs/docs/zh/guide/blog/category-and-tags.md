可以通过 frontmatter 来为文章配置分类与标签，使它们出现在特定的分类与标签页面中。

## 分类设置

只需要在页面的 frontmatter 中设置 `categories` 数组，并设置一个或多个文章所属的分类，
该文章会自动渲染到 `/categories/<分类名>` 分类页面的列表中。

比如在某个页面设置：

```md
---
categories:
  - Vue
  - JavaScript
---

页面内容...
```

## 标签

只需要在页面的 frontmatter 中设置 `tags`，并设置一个或多个文章所属的标签，该文章会自动渲染在
`/tags/<标签名>` 标签页面的列表中。

比如在某个页面设置：

```md
---
tags:
  - Vue
  - JavaScript
---

页面内容...
```

## 查看列表

除了直接访问对应的低之外，分类和标签名称会显示在文章结尾的信息处，你可以点击它来导航到对应的列表查看相同分类或标签的文章。
