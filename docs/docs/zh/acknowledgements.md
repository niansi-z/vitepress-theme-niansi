# 项目参考

本项目在开发过程中参考了多个优秀的开源项目，以下是主要的参考来源：

## Mister-Hope 的 VuePress 主题

[Mister-Hope](https://github.com/Mister-Hope) 的 VitePress 主题是本项目最重要的参考来源之一。

### 参考内容

- **主题配置结构**：参考了其配置文件的组织方式和主题配置选项的设计
- **Markdown 扩展**：借鉴了其对 Markdown 语法的高級扩展，如上下角标、标记等特性
- **提示容器**：参考了其提示容器（Containers），提供了多种样式的提示框
- **预览容器**：参考了其预览容器（Preview）的设计
- **任务列表**：参考了其任务列表（TaskList）的实现

## Chirpy Jekyll 主题

[Jekyll Chirpy](https://github.com/cotes2020/jekyll-theme-chirpy) 是一个简洁优雅的 Jekyll 博客主题，本项目借鉴了其许多设计理念。

### 参考内容

- **博客功能设计**：借鉴了其博客功能的设计思路，包括文章布局、元数据展示等
- **页面设计**：借鉴了其页面的设计和实现

## VitePress 官方功能

[VitePress](https://vitepress.dev/) 是 VuePress 的继任者，本项目基于 VitePress 构建，充分利用了其核心功能。

### 使用的基础功能

- **导航栏**：基于 VitePress 的导航栏功能
- **侧边栏**：基于 VitePress 的侧边栏功能
- **Markdown 渲染**：基于 VitePress 的 Markdown 处理
- **主题扩展**：使用 VitePress 的主题扩展机制
- **本地搜索**：集成 VitePress 的本地搜索功能

### 项目地址

- 官方文档: [VitePress](https://vitepress.dev/)
- GitHub: [vuejs/vitepress](https://github.com/vuejs/vitepress)
