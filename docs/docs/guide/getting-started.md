## 安装

参考 VitePress 的安装要求。

::: code-group

```sh [npm]
$ npm add -D vitepress-theme-niansi@next
```

```sh [pnpm]
$ pnpm add -D vitepress-theme-niansi@next
```

```sh [yarn]
$ yarn add -D vitepress-theme-niansi@next vue
```

```sh [bun]
$ bun add -D vitepress-theme-niansi@next
```

:::

## 安装向导

VitePress-Theme-Niansi 附带一个类似 VitePress 的命令行设置向导，可以帮助你构建一个基本项目。安装后，通过运行以下命令启动向导：

::: code-group

```sh [npm]
$ npx vitepress-theme-niansi init
```

```sh [pnpm]
$ pnpm vitepress-theme-niansi init
```

```sh [yarn]
$ yarn vitepress-theme-niansi init
```

```sh [bun]
$ bun vitepress-theme-niansi init
```

:::

::: tip
问题参考 VitePress 的安装向导，但是移除了 JS 的选项。
:::
