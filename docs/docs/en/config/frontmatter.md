You can set information for page with the following frontmatter options.

## title

- Type: `string`
- Required: No

Current page's title. Markdown's first `h1` by default.

## excerpt

- Type: `string`
- Required: No

Current page's excerpt.

## author

- Type: `string`
- Required: No

Current page's author.

## date

- Type: `DateString`
- Required: No

Writing time, format: `YYYY-MM-DD`.

## categories

- Type: `string | string[]`
- Required: No

Categories.

## tags

- Type: `string | string[]`
- Required: No

Tags.
