The project has built-in footnote support, allowing you to add footnotes directly in your Markdown files.

## Demo

#### Input

```md
Here’s an example of a footnote[^1].

[^1]: This is the footnote content.
```

#### Output

Here’s an example of a footnote[^1].

[^1]: This is the footnote content.
