## Usage

You can include another Markdown file within a Markdown file like this, or even embed it inline.

:::tip Tips
You can also use `@`, which points to the source root directory. By default, this is the VitePress project root, unless `srcDir` is configured.
:::

For usage details, see [VitePress → Markdown File Inclusion](https://vitepress.dev/zh/guide/markdown#markdown-file-inclusion)。
