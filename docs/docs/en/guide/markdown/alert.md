The theme adds GFM alerts support.

## Demonstration

#### Input

```md
> [!NOTE]
> This is note text

> [!TIP]
> This is tip text

> [!IMPORTANT]
> This is important text

> [!WARNING]
> This is warning text

> [!CAUTION]
> This is caution text
```

#### Output

> [!NOTE]
> This is note text

> [!TIP]
> This is tip text

> [!IMPORTANT]
> This is important text

> [!WARNING]
> This is warning text

> [!CAUTION]
> This is caution text
