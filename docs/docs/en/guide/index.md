## Introduction

VitePress Theme Niansi is a VitePress theme with a rich set of features. Of course, various features are still under development.

First, you should already have a basic understanding of the following:

- [Markdown](https://www.markdownlang.com/zh/)（including the concept of [YAML Frontmatter](../get-started/content#Frontmatter%20Configuration)）
- [VitePress](https://vitepress.dev/guide/what-is-vitepress)

:::tip Tips
- If you are completely new to VitePress, Markdown, or other related content, please start with the [Getting Started](../get-started/index.md).
- If you are familiar with the above, you may [start here](./intro/intro.md)
:::
