You can control the paths of each page in the blog through theme options.

The default paths provided by the blog are as follows.
If they conflict with your existing paths and you do not want to adjust your own paths, you can modify them.

| **Options**        | **Description** | **Default Path**       |
|----------------|-----------------|----------------|
| `archivesPath` | Articles page   | `/archives/`   |
| `categoryPath` | Category page   | `/categories/` |
| `tagPath`      | 	Tag page       | `/tags/`       |
