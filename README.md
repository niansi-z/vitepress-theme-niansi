## 在线安装

```bash
pnpm add -D vitepress-theme-niansi vitepress@next
```

```bash
pnpm vitepress-theme-niansi init
```

## 本地运行

项目拉取

```bash
git clone https://github.com/niansi-z/vitepress-theme-niansi.git
```

依赖安装

```bash
pnpm install
```

运行

```bash
pnpm theme:dev
```
