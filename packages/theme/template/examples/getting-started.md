## Quick Start

```bash
pnpm add -D vitepress-theme-niansi vitepress@next
```

## Create Config

```js
import { defineConfig } from 'vitepress-theme-niansi'

export default defineConfig({
  title: 'My Blog',
  themeConfig: {
    author: 'Your Name',
  }
})
```
